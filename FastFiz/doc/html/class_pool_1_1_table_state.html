The physical state of balls on a table.

Includes position and velocity information but no rules related information.

bool  verbose = false,

Execute given shot on the table state.

The TableState object will be modified to reflect the final state of the shot. If you need to retain the initial state, copy the object before calling. Noise is not added to the shot prior to execution.

Parameters
spThe shot parameters to execute.
verbosePrint a lot of debugging information to cerr.
errorsDetect internal errors (slower execution)
Exceptions
BadShotExceptionwill be thrown in case of bad parameters or errors in execution.
