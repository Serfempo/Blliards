Detailed Description

This defines most of the information and functions necessary about the state of a game.

All rules objects subclass off this.

enum Pool::GameState::PreProcessCode
protected

Possible results of pre-evaluation of shot parameters for current game state.

Enumerator
PPC_BADPARAMS 

Shot parameters not satisfactory for game state.

PPC_G_NOEXECUTE 

Decision information in shot is sufficient.

No need to execute shot.

PPC_G_NORMAL 

Shot should be processed normally.

Cue ball is not to be moved.

PPC_G_PLACECUE 

Shot should be processed normally.

Cue ball is to be placed at requested position.

Optionally, returns the Shot object used to execute the physical shot.
